# UI Theme Redesign (Premium Light Mode)

Replace current dark theme with premium light theme.

Goal:

Create clean, modern, premium UI similar to Apple. Focus readability, whitespace, subtle depth, elegant animations.

Do NOT simply invert colors. Redesign entire color system.

## Glassmorphism

Keep glass effect.

Reduce blur strength.

Avoid overly transparent cards.

Use soft shadows instead.

Example

- blur 12-20px
- opacity 85-95%
- subtle border
- soft shadow

Cards should feel like floating paper, not frosted glass everywhere.

---

## Shadows

Use soft premium shadows.

## Overall Goal

Website should feel premium from first impression.

Think Apple, not generic admin dashboard.

Prioritize

- readability
- whitespace
- consistency
- elegance
- subtle depth
- premium polish

Do NOT change functionality.

Only redesign visual system while preserving existing components, workflows, responsiveness, animations, APIs, database, routing.
